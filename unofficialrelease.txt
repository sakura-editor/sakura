本リリースは「たけ」さんのテキストエディタSAKURAの公開ソース(2000/02/21付)を
元に不具合修正、機能追加を行ったものです。

●著作権
このプログラムの著作権は各箇所を記述した人にあります。記述者が特に明示されて
いない箇所はファイル先頭のAuthorが著作権を保持しています。
ただし、ビットマップに関してはたけ(nakatani)，じぇぷろ，hor，26(@2ch.net)
(敬称略)が、アイコン、カーソルに関してはたけ氏が著作権を保持しています。
また自動的に作成されるファイルの著作権はその主たるファイルの著作権者が
保持しています。

●ライセンス
「げんた」「やざき」「みく」「すとーにぃ」作の部分に関してはzlib/libpng
ライセンスに従って著作権表示の削除と著作者の詐称を除くあらゆる利用，配布が
許可されています。

その他の部分についてはサクラエディタの開発を目的とした場合に限り変更部分を
含めたソースコードを開示することを条件に変更，流用が認められています．
その他の目的での利用は各著作権者に個別に問い合わせていただくことになります．

なお、テキストファイル(.txt)は作成者が共同で著作権を保持しているものとします。
このソフトウェアの配布精神に反しない範囲で自由な改造及び再配布が可能です。

●制限事項
このプログラムをコンパイルするためには以下に示すソフトウェア群のうちいずれかが必要です．
(a) Microsoft Visual C++ 6.0 + HTML Help Workshop
(b) Microsoft Visual C++ 6.0 + 最新のPlatform SDK
(c) Borland C++ Compiler 5.5.1
