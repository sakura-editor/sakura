Sakura-Editor Macro Reference

S_ChgCharSet

機能
文字コードセット指定
説明
ファイルの文字コードセット指定とBOMの有無を変更します。次回保存時からファイルに適用されます。
構文
void S_ChgCharSet ( i1 ， i2 )

無題
引数型内容
i1整数数値文字コード（省略可能）
i2整数数値BOMの有無（省略可能）
設定値
引数 i1 の設定値は次のとおりです。

無題
値内容
-1ダイアログを表示（省略時規定値）
0SJIS
1JIS
2EUC
3Unicode
4UTF-8
5UTF-7
6UnicodeBE
7CESU-8
8Latin1(Windows-1252)
99文字コードを維持

引数 i2 の設定値は次のとおりです。

無題
値内容
0BOMなし（省略時規定値）
1BOMあり(0以外)
