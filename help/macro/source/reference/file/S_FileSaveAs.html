Sakura-Editor Macro Reference

S_FileSaveAs

機能
名前を付けて保存
構文
void S_FileSaveAs ( S1 ， i1 ， i2 )  使用例

引数型内容
S1文字列保存するファイル名
i1整数数値文字コードセット
i2整数数値改行コード
設定値
引数 i1 の設定値は次のとおりです。

値内容
0SJIS
1JIS
2EUC
3Unicode
4UTF-8
5UTF-7
6UnicodeBE
7CESU-8
8Latin1(Windows-1252)
99文字コードを維持

引数 i2 の設定値は次のとおりです。

値内容
0変更しない
1CRLF
2LF
3CR
補足
PPAのマクロ中では $41 などの 16進数表記や変数・数式・関数の使用も可能だが、MACのマクロ中では 65 などの 10進数の定数表記しか受け付けない。
