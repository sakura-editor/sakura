Sakura-Editor Macro Reference

マクロの種類

拡張子により、実行するエンジンを識別します。拡張子は正しくつけて下さい。

キーマクロ（キーボードマクロ）

拡張子
mac
対応バージョン
0.3.7.2以降
説明
一部の関数が使えない。
[キーマクロの保存]でも作成できる。

ＰＰＡマクロ

拡張子
ppa
対応バージョン
1.2.106.1以降
説明
別途ppa.dllが必要
実行環境に左右されず一定の動作が期待できる

(WSH)JScript/VBScript

拡張子
js/vbs/
対応バージョン
1.3.3.0以降
Win95系OSは1.4.3.4以降
説明
Win95系OSでは動作が微妙

(WSH)Perl/Python/Rubyその他

拡張子
pls/pys/...
対応バージョン
1.3.3.0以降
Win95系OSは1.4.3.4以降
説明
Win95系OSでは動作が微妙
対象の言語エンジンが必要。
