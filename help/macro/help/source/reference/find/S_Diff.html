Sakura-Editor Macro Reference

S_Diff

機能
DIFF差分表示
構文
void S_Diff ( S1, i1 )

引数型内容
S1文字列比較ファイル名
i1整数数値DIFF ダイアログの状態を10進数数値で指定する。
設定値
引数 i1 の設定値は次のとおりです。

それぞれの bit が 0=チェックOFF / 1=チェックON
bit内容
0(-i) 大文字小文字同一視
1(-w) 空白無視
2(-b) 空白変更無視
3(-B) 空行無視
4(-t) TAB-SPACE変換
5編集中のファイルが旧ファイル
6DIFF差分がないときにメッセージ表示
7- No Use -
